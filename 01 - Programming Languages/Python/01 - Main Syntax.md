# Main Syntax
## IF...Else Condition
```python
if <condition1> and <condition2> or <condition3> not <condition4>:
	<instruction1>
	<instruction2>
	<instruction3>
	...
```

## While Loop
```python
while <condition1> and <condition2> or <condition3> not <condition4>:
	<instruction1>
	<instruction2>
	<instruction3>
	...
```

## For Loop
```python
for <value> in <condition>:
	<instruction1>
	<instruction2>
	<instruction3>
	...
```
