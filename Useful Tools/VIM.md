# VIM Cheatsheets
## Command Useful
### Number Line
** Config Number Line in the follow command **
```vim
:set number
```

### Tab Indent
** Config Tab Indent in the follow command  **
```vim
:set shiftwidth=4
:set sw=4
```

### Folding
** Config Fold Method in the follow command **
```vim
:set foldmethod=indent
```

** Operate the Indents **

| Command | Value |
| ------- | ----- |
| zc | Indent Single Close |
| zC | Indent All Close |
| zo | Indent Single Open |
| zO | Indent All Open |
